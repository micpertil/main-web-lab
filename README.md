# main-web-lab

## 项目简介

这是一个用于归档和存放多个独立 HTML 页面的实验室仓库。仓库中的页面涵盖不同类型的展示、测试和实验内容，主要用于个人学习、前端练习以及小型项目的快速原型展示。

本仓库不针对任何特定网站或域名，所有页面均为独立运行、不依赖外部服务的静态 HTML 文件。

## 目录结构

```
main-web-lab/
├── index.html          # 仓库页面导航首页
├── page-xxx/           # 各独立页面目录（按功能或主题命名）
├── assets/             # 共享资源文件（CSS、JS、图片等）
└── README.md           # 本说明文档
```

- **根目录**：包含仓库主入口文件 `index.html`，用于快速浏览所有归档页面。
- **page-xxx**：每个子目录存放一个独立页面及其相关资源，目录名反映页面主题或用途。
- **assets**：存放多个页面可能共用的资源，如通用样式、脚本或占位图片。

## 页面归档说明

本仓库中的页面均以**独立、自包含**为原则归档。每个页面均可直接通过浏览器打开运行，无需额外服务或依赖。

- 所有页面均为纯前端 HTML/CSS/JavaScript 实现。
- 页面中不包含任何外部链接、追踪代码或商业推广内容。
- 部分页面可能包含实验性功能或未完成的部分，仅供学习参考。

## 维护说明

本仓库由个人维护，不定期更新。页面归档遵循以下原则：

- 新增页面时，在 `index.html` 中添加导航链接。
- 保持目录结构清晰，避免资源冗余。
- 不接收第三方提交的页面或外部链接。
- 如发现页面功能异常，可提交 Issue 说明，但无法保证即时修复。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接用浏览器打开 `index.html` 查看页面列表。
3. 点击对应链接进入各独立页面。
